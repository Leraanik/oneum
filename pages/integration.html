Ваши интеграции

  • Введите в поле поиска @BotFather и выберите бота.

  • Нажмите «Запустить» для активации бота BotFather.

  • Выберите или напечатайте и отправьте команду /newbot.

  • Дайте имя боту — клиенты увидят это имя при общении с ботом. Придумайте никнейм бота —
    по нему можно будет найти бота в Telegram. Никнейм должен быть уникальным, не повторять
    существующие в базе и заканчиваться на слово «bot».

  • Скопируйте HTTP API в форму выше.
